# Portafolio Web Profesional - Mark Orsini

Portafolio web desarrollado para la asignatura de Programación Web del Liceo Comercial Vate Vicente Huidobro. Este proyecto demuestra mis habilidades en desarrollo web, diseño responsivo y lógica de programación.

[Ver en línea](https://Mark-Orsini.github.io/Portafolio)

## Descripción

Sitio web moderno y responsivo que presenta mi perfil profesional, proyectos y habilidades técnicas. Diseñado con un enfoque en la experiencia de usuario y la calidad del código.

## Características Principales

- **Diseño Responsivo**: Adaptable a dispositivos móviles y de escritorio.
- **Multiidioma**: Soporte para múltiples idiomas (Español, Inglés, etc.).
- **Personalización**: Múltiples temas de color y configuraciones de usuario.
- **Interactividad**: Navegación fluida y elementos dinámicos.

## Tecnologías

- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **Bootstrap 5**

## Estructura del Proyecto

```
/
├── index.html          # Página principal
├── settings.html       # Configuración
├── static/
│   ├── css/            # Estilos (styles.css, themes.css)
│   ├── js/             # Lógica (main.js, translations.js)
│   └── img/            # Recursos gráficos
└── README.md
```
